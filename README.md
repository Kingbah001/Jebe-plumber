# JEBE Plumbing Tech - High-Converting Landing Page

Modern, responsive landing page for Lagos-based plumbing company.

## Features
- Hero section with CTAs
- Services grid with icons
- Testimonials carousel
- Contact form + WhatsApp
- Google Maps integration
- Mobile-first responsive
- Smooth animations

## Quick Start
```bash
start index.html
```
Opens in default browser.

## Customization
- Replace Unsplash images with real project photos
- Update Formspree endpoint in script.js
- Add Google Analytics

Built with HTML5, CSS3, Vanilla JS. No frameworks needed.
